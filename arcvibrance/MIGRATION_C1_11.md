# ArcVibrance C1.11

This UI refinement focuses on the profile editor.

- Rebuilt the saturation numeric editor as one fixed rounded surface.
- Centered the increment and decrement chevrons in equal-height rows.
- Stopped rewriting text while the user types, preserving caret and selection behavior.
- Values are committed and clamped to the supported 0–300 range.
- Added a clearly labelled placeholder for a future per-profile custom resolution override.
- Realigned the Cancel and Save Changes buttons into a consistent bottom action row.
- Native agent, IPC, profiles, monitoring, and packaging remain unchanged.
