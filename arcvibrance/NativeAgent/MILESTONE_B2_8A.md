# Milestone B2.8A — Buttons and profile cards

This revision is based on the stable B2.7 branch.

Changes are intentionally limited to presentation:
- rounded primary and secondary owner-drawn buttons
- blue-to-purple primary button gradient
- improved hover, pressed, focus, disabled, and destructive states
- profile list items rendered as spaced rounded cards
- selected profile receives an accent outline
- slightly increased list-card spacing and action-row spacing

No slider, numeric input, toggle, profile persistence, IPC, or agent behavior was changed.
