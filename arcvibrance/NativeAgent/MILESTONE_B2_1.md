# Milestone B2.1 - Enhanced profile manager

- Added persistent enabled/disabled state for each game profile.
- Added an Enable / disable action to the profile manager.
- Disabled profiles remain visible but are ignored by the background runtime.
- Updated the profile format to version 2 with backward-compatible loading of version 1.
- Renamed Add game to Browse game; executable selection, automatic executable-name detection, and duplicate-path protection remain active.
