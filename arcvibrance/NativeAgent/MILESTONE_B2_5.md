# B2.5 — Neon UI overhaul, phase 1

This milestone applies the approved dark blue / cyan / violet visual direction to the native Win32 application while leaving the runtime, agent, IPC, profile storage, Intel backend and matching logic unchanged.

## Changes

- Wider 1040×820 dashboard layout
- Branded left sidebar and monitoring card
- Larger profile workspace and card-style rows
- 36 px executable icons
- Prominent Add game and profile action controls
- Consolidated Quick settings panel
- Darker navy palette with brighter cyan accents
- Larger, cleaner profile editor with exact numeric input and Save changes action
- Existing System / Light / Dark setting remains available

This is phase 1 of the visual overhaul. It intentionally stays on native Win32 controls to avoid destabilizing the working backend.
