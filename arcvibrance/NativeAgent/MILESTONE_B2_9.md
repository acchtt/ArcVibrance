# ArcVibrance B2.9

- Adds a dedicated **Profiles** navigation button above **Settings**.
- Draws a controller icon for Profiles and a gear icon for Settings using anti-aliased vector graphics.
- Keeps the cyan-violet-magenta gradient on the active navigation item.
- Removes the border/outline from inactive navigation items while retaining a subtle hover fill.
- Assigns unique navigation control IDs so Settings controls can no longer be mistaken for sidebar clicks.
- Replaces the native saturation edit field and detached `%` label with one custom rounded numeric spinner.
- Centers the numeric value and `%` in the same control and adds divided up/down arrow buttons.
- Supports mouse clicks, press-and-hold repeat, mouse wheel, arrow keys, Page Up/Down, Home/End, and direct numeric typing.
- Keeps the spinner and slider synchronized and clamps saturation to 0–300%.
