# ArcVibrance C1.7 UI refinement

- Reduced default window size to 1280x820.
- Removed the Profiles-page Quick Actions / summary strip.
- Replaced sidebar ToggleButtons with centered transient-state navigation buttons.
- Added a subtle active-page indicator instead of a persistent bright pressed state.
- Replaced the NumberBox overlay with a custom numeric saturation editor.
- Added typing validation, clamping, slider synchronization, keyboard arrows, Enter commit, and spin buttons.
- Removed Profile Status from the Edit Profile panel; enable/disable remains available on each profile card.
